# cn-portal-resource

本项目用于归档和发布多个独立的 HTML 页面资源。仓库内容不针对任何特定域名或具体网站，仅作为通用资源存储与分发用途。

## 项目简介

cn-portal-resource 是一个静态资源仓库，主要存放各类独立 HTML 页面。这些页面可被单独访问或嵌入其他项目中，用于展示、测试或信息归档等目的。仓库不依赖任何外部服务或后端，所有资源均为纯前端 HTML 文件。

## 目录说明

```
cn-portal-resource/
├── pages/          # 独立 HTML 页面文件
├── assets/         # 公共资源（CSS、JS、图片等）
├── docs/           # 文档或说明文件（可选）
└── README.md       # 本文件
```

- `pages/`：存放每个独立的 HTML 页面，每个页面通常为一个自包含的完整文件。
- `assets/`：存放页面之间可能共享的样式、脚本或媒体文件。
- `docs/`：用于放置与仓库相关的额外文档或说明。

## 页面归档说明

- 每个 HTML 页面应尽量保持独立，减少对外部资源的强依赖。
- 页面命名应简洁明了，反映其内容或用途。
- 如有引用公共资源，请统一放置在 `assets/` 目录下，并在页面中使用相对路径引用。
- 所有页面均为静态资源，不涉及用户数据收集或服务端处理。

## 维护说明

- 欢迎提交新的独立 HTML 页面或改进现有页面。
- 提交前请确保页面无外部链接依赖（除非必要且稳定）。
- 请勿在页面中包含任何具体域名、网址、营销信息或诱导点击的内容。
- 如有问题或建议，可通过 Issues 进行讨论。
